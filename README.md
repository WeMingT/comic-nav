# 漫画资源导航 (Comic Resources Navigation)

这是一个基于知乎回答「[漫画资源应该去哪里找？](https://www.zhihu.com/question/48787704)」整理而成的静态导航页面。

本项目旨在为漫画爱好者提供一个清晰、便捷的资源索引，涵盖了正版平台、免费汉化资源、生肉资源、阅读工具（Mihon/Suwayomi）及其插件推荐等多种途径。

## ✨ 特性

- **🌓 深色护眼主题**：默认采用深色配色，适合长时间浏览。
- **📱 响应式设计**：完美适配移动端（手机/平板）与桌面端显示。
- **📂 清晰分类**：资源按类型（网盘搬运、正版、汉化、生肉、英文、插件等）结构化展示。
- **🧩 插件推荐**：专门的板块展示 Mihon/Suwayomi 的常用插件及配置源。
- **🔍 交互体验**：使用折叠面板隐藏长文本说明，保持页面整洁。

## 🚀 使用方法

### 本地浏览

1. 下载本项目代码或 `git clone` 到本地：

   ```bash
   git clone https://github.com/your-username/comic-nav.git
   ```

2. 直接双击打开 `index.html` 文件即可在浏览器中预览。

### 🌐 部署 (推荐 GitHub Pages)

你可以轻松地将此页面免费托管在 GitHub Pages 上：

1. 将本项目推送到你的 GitHub 仓库。
2. 进入仓库的 **Settings** (设置) 页面。
3. 在左侧菜单找到 **Pages**。
4. 在 **Build and deployment** -> **Branch** 中选择 `main` 分支，点击 **Save**。
5. 等待片刻，GitHub 会生成一个类似 `https://<你的用户名>.github.io/comic-nav/` 的访问链接。
